Jakob Frick

Developer and Aspiring Innovator and Educator

About Me...

Who am I?
I'm Jakob Frick, a 19-year-old passionate developer from Sweden.

Why do I write code?
I've always strived to understand how things work. Whether it be in math, physics, or electronics, finding the reason behind why stuff works the way it does is very interesting to me, and when I first found the game Minecraft back in 2010, I immediately wanted to know what was going on behind the scenes.

Where did I start?
I learned some basic Minecraft-environment specific Java with the help of various modding forums, I managed to make a few mods, long since buried into obscurity. Despite the time I put into my mods, I never properly learned object orientation, actual Java features, or even basic programming. All I knew was Minecraft modding.

How did I develop my skills?
Turns out, there's a mod for Minecraft called ComputerCraft. It adds computers to Minecraft, with a LUA-based operating system integrated into them. This allows you to write your own programs and robots to perform various tasks for you. This is where I learned all about loops, arrays, functions, recursion, and a bunch of other cool basics. The pilgrimage from the whitespace based syntax in LUA to Javascript syntax, was a long, confusing and frustrating one, and the advance to object oriented programming with C#, even more so.
